Регистрация

При регистрации пользователю даётся выбор из классической регистрации через сервис или через социальную сеть.

При классической регистрации пользователь обязан ввести свой username только в маленьком регистре и латинскими буквами иначе вылезет ошибка ( это сделано для того, чтобы после успешной регистрации дать новому пользователю уникальную страницу, которую можно найти по его username в url ).

При классической регистрации на почту будет высланно письмо с подтверждением указанной раннее почты.

Авторизация

Нужно только корректно ввести свой логин и пароль. Если пользователь забыл пароль, то он сможет сгенерировать новый, перейдя по ссылке 'Забыли пароль?'.

Опять же, существует вход через социальную сеть для упращения процесса.

Панель управления / Создание карточки

Пользователь имеет возможность создать свою карточку и добавить:

  • Аватарку
  • Заголовок
  • Описание
  • Большую кнопку прямого звонка
  • Слайдер быстрых ссылок на соцсети, мессенджеры, почту и другое
  • Галерею видео с youtube
  • Галерею картинок
  • Таблицу цен с наименованиями и категориями
  • Частые вопросы и ответы на них
  • Изменить задний фон своей страницы

Принцип такой, что нужно создать блок, а затем уже выбрать какой модуль будет находиться в блоке.

Все модули и блоки можно назвать как хочется, менять их местами в очереди и при желании всё редактируется (цвета, иконки, названия, порядок и т.д.).

Основная мысль и идея блоков в том, что их можно свернуть для экономии места. Для этого в панеле управления при создании блока можно выбрать опцию 'Свернуть'.
